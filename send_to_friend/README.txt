HOW TO INSTALL:
-----------------------------

# Unzip the file
# Upload the "send_to_friend/" folder to the website (eg: confor.tv/send_to_friend/)
# Add this script tag to confor.tv:

# This one have to be genarated dynamicaly after language detection
# So with or without _fr
<script language="javascript" src="/send_to_friend/popup.php?js" type="text/javascript"></script>

# Create a link under all videos:
# This one have to be genarated dynamicaly after language detection
# So with or without _fr
<a href="/send_to_friend/popup.php" title="Send to Friend" class="sendtofriend"><img border="0" src="assets/img/sendtoafriend.png" /></a>

# Done!
